    Descripción:

    Esta pagina web es un proyecto educativo de la asignatura desarrollo web. Es una pagina desarrollada con html, javascript para la funcionalidad de la pagina, css y con un frameworks de css como bootstrap para el diseño. Es una pagima web de Ingeniería de sistemas de la Universidad de Medellín, En la cual se muestran noticias relacionadas con tecnología, empresas de tecnología y ofertas de empleo de esas empresas. También tiene implementación de un Login de usuarios.

    Objetivos:
    • Poner en practica los conceptos de desarrollo web vistos en clase.
    • Practicar el manejo de algun framework de css.
    • Practicar el uso e implementación de CRUDs por medio de archivos json en el desarrollo web.
    • Sacar 5.0 en la nota del proyecto de aula parte 1, ya que es el mejor proyecto.
